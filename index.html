About

Forex ATS is an automated trading system for the foreign exchange market. To keep it on the cutting edge, we run regular competitions where developers submit improved versions of the algorithm. The best entry becomes the new master strategy and wins the prize.

How It Works

  1. Register for the next competition.
  2. Improve the current trading algorithm by adding features or optimisations.
  3. Submit your version for evaluation in a live trading environment.
  4. The version with the best performance wins and becomes the new master strategy.

Cash prizes are funded by an optional copy‑trading service that replicates the master algorithm across user accounts. As performance improves and more users join, the prize pool grows for the next round.

Why Participate?

  • Sustainable funding: prize pools come from copy‑trading fees, not external sponsors.
  • Continuous improvement: every winning entry strengthens the algorithm and benefits all users.
  • Growing community: better performance attracts more copy traders, increasing the next prize pool.
  • Open innovation: anyone can compete and contribute fresh ideas.
